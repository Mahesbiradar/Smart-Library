Project Overview

Smart Library is a comprehensive digital library management system designed to streamline book issuing, returning, and inventory management processes. Our system provides an intuitive interface for students and administrators with features like barcode scanning, CSV import/export, and real-time transaction tracking.


This project demonstrates our expertise in full-stack web development, database management, and user experience design.
